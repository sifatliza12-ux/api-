# ForgeFlow Backend

A clean, modular Node.js + Express starter backend for the ForgeFlow Chrome extension.

## Features

- Express server running on port 5000
- CORS configured for future Chrome extension communication
- Modular routing for authentication, APIs, marketplace, and subscription
- In-memory sample data for initial development
- Error handling middleware
- TODO placeholders for future database, authentication, and payment integration

## Scripts

- `npm install`
- `npm run dev`
- `npm start`
